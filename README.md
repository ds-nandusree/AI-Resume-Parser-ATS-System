# 🚀 AI Resume Parser & ATS Analyzer

An intelligent recruitment assistance platform that automates resume screening using Machine Learning, Natural Language Processing (NLP), Semantic Similarity Matching, and ATS (Applicant Tracking System) scoring.

The system extracts candidate information from resumes, identifies technical skills, evaluates job-description compatibility, performs semantic skill matching, analyzes skill gaps, generates ATS scores, and produces downloadable PDF reports for recruiters and job seekers.

## 🌐 Live Demo

**Application:** https://ai-resume-parser-ats-system-fbs6ozg98uruuuokzxqovo.streamlit.app/

## 🎯 Problem Statement

Traditional resume screening is time-consuming and often fails to identify suitable candidates efficiently. Recruiters spend significant effort manually reviewing resumes against job requirements.

This project addresses the problem by automating resume analysis and providing data-driven candidate evaluation through AI-powered matching and analytics.

---

## ✨ Key Features

### 📄 Resume Parsing

* Extracts text from PDF resumes
* Processes structured and unstructured resume formats
* Supports automated resume analysis

### 🎯 Role Prediction

* Predicts the candidate's professional category using a trained Machine Learning model
* Identifies the most relevant career domain

### 🛠 Skill Extraction

* Detects technical skills from resumes using NLP techniques
* Matches extracted skills against a curated skills database

### 📧 Contact Information Extraction

* Email Detection
* Phone Number Detection
* LinkedIn Profile Detection

### 📊 ATS Score Calculation

* Calculates compatibility between resume and job description
* Provides a realistic ATS-style evaluation score

### 🧠 Semantic Skill Matching

* Uses Sentence Transformers for contextual similarity analysis
* Detects related skills even when exact keywords are absent

### 💪 Resume Strength Analysis

* Evaluates overall resume quality
* Generates a Resume Strength Score

### 📉 Skill Gap Analysis

* Identifies missing skills required for a target role
* Calculates skill-gap percentage

### 📚 Learning Recommendations

* Suggests the most important skills to acquire
* Helps candidates improve job readiness

### 📈 Interactive Dashboard Analytics

* Visualizes skill matching results
* Displays ATS insights through interactive charts

### 📄 PDF Report Generation

* Generates downloadable ATS evaluation reports
* Summarizes strengths, weaknesses, and recommendations
